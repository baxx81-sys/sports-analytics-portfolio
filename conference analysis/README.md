Power Conference Efficiency Analysis

This project analyzes performance across major college football conferences:

* SEC
* Big 12
* Big Ten
* ACC

Objective

To determine which conferences produce the most efficient offensive and defensive teams.

Metrics Used

* Points Per Game (Offensive Production)
* Points Allowed (Defensive Performance)
* Offensive Power Score
* Defensive Power Score

Key Insight

This analysis explores whether certain conferences are more offense-driven or defense-driven.

Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
